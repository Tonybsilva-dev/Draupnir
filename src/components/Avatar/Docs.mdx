# Avatar Component
The `Avatar` component is used to represent a user, and can display images, icons or initials. It is commonly used in user interfaces to signal the presence of a person or entity.

## Features
- Supports images, icons and initials
- Customizable in size and shape
- Fallback support for initials when image is not available
