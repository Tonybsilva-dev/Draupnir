# Switch Component
The `Switch` component is used to represent a user, and can display images, icons or initials. It is commonly used in user interfaces to signal the presence of a person or entity.

## Features
- Toggles state between enabled and disabled with a simple click.
- Available in two variants: "common" for a simple switch, and "contract" for a switch with checkmark and cross icons.
- Supports default enabled state configuration through defaultEnable.
- Allows custom behavior on state change via onChange callback.
- Adaptable styling with useStyleSwitch for dynamic appearance based on state and variant.
- Accessibility features with hidden descriptive text for screen readers.
