# Text Block

O componente Text Block é utilizado para exibir blocos de texto com título e descrição, facilitando a organização de informações.

## Descrição
O Text Block do Draupnir DS destaca textos importantes, títulos de seções e descrições de forma clara e acessível.

## Recursos
- **Acessível:** Tipografia semântica, foco visível, estrutura clara
- **Customizável:** Permite título, descrição e children
- **Responsivo:** Adapta-se a diferentes tamanhos de tela

## Uso

```tsx
import { TextBlock } from 'draupnir-ds';

<TextBlock title="Título" type="primary">
  Descrição ou conteúdo do bloco de texto.
</TextBlock>
```
