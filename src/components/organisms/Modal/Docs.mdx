# Modal

O componente Modal exibe uma janela sobreposta para destacar informações ou ações importantes.

## Descrição
O Modal do Draupnir DS oferece foco gerenciado, fechamento por ESC, animações suaves e integração com formulários.

## Recursos
- **Acessível:** Trava foco, ARIA, fechamento por ESC, título e descrição para leitores de tela
- **Modal e não-modal:** Suporte a ambos os modos
- **Animado:** Framer Motion e Tailwind para transições
- **Controlável:** Pode ser controlado por estado externo
- **Customizável:** Cabeçalho, corpo e rodapé customizáveis

## Uso

```tsx
import { Modal, ModalBody, ModalFooter, ModalHeader } from 'draupnir-ds';

<Modal isOpen={isOpen} onClose={() => setIsOpen(false)}>
  <ModalHeader>Título</ModalHeader>
  <ModalBody>Conteúdo do modal</ModalBody>
  <ModalFooter>
    <Button>Salvar</Button>
    <Button variant="outline">Cancelar</Button>
  </ModalFooter>
</Modal>
```
