# Modal
This is a modal component.

## Feature
- Modal/Non-Modal Modes: Supports both focused modal and non-intrusive non-modal usage.
- Focus Management: Automatically traps focus in modal mode for enhanced accessibility.
- Control Flexibility: Can be used in both controlled and uncontrolled manners.
- Accessibility Features: Integrates with Title and Description components for screen reader announcements.
- Esc Key Closure: Enables intuitive dismissal with the Esc key.
- Animations and Variants: build with Framer Motion and Tailwindcss pure.
