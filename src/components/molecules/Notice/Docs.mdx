# Notice

O componente Notice é utilizado para exibir mensagens de alerta, sucesso, erro ou informação para o usuário.

## Descrição
O Notice do Draupnir DS oferece feedback visual imediato, com ícones, cores e possibilidade de fechamento.

## Recursos
- **Tipos:** Success, Alert, Error, Info
- **Ícones automáticos:** Exibe ícone conforme o tipo
- **Acessível:** Suporte a ARIA, navegação por teclado, foco visível, botão de fechar acessível
- **Customizável:** Permite adicionar conteúdo customizado
- **Interativo:** Pode ser fechado pelo usuário (botão ou tecla Esc)

## Uso

```tsx
import { useNotifications, NotificationsProvider, NotificationsContainer } from 'draupnir-ds';

function MeuComponente() {
  const { notify } = useNotifications();

  return (
    <button
      onClick={() =>
        notify({
          notificationType: 'notice',
          type: 'success',
          message: 'Operação realizada com sucesso!',
          position: 'center-top',
          autoCloseMs: 3000,
        })
      }
    >
      Mostrar Notice
    </button>
  );
}

// No seu App:
<NotificationsProvider>
  <NotificationsContainer />
  <MeuComponente />
</NotificationsProvider>
```
