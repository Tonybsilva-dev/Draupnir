# Subtitle Page

O componente Subtitle Page exibe um subtítulo, descrição e ações no topo de seções da interface.

## Descrição
O Subtitle Page do Draupnir DS organiza informações secundárias e botões de ação, melhorando a hierarquia visual.

## Recursos
- **Acessível:** Tipografia semântica, foco visível, estrutura clara
- **Responsivo:** Layout adaptável para diferentes tamanhos de tela
- **Customizável:** Permite adicionar botões, descrições e outros elementos

## Uso

```tsx
import { SubtitlePage } from 'draupnir-ds';

<SubtitlePage subtitle="Informações pessoais" description="Atualize seus dados aqui.">
  <Button variant="outline">Cancelar</Button>
  <Button variant="primary">Salvar</Button>
</SubtitlePage>
```
