# Input

O componente Input é utilizado para capturar e submeter dados do usuário.

## Descrição
O Input do Draupnir DS oferece campos de texto com prefixo, estados de erro, desabilitado e integração com formulários.

## Recursos
- **Prefixo:** Suporte a ícones ou texto antes do campo
- **Estados:** Normal, erro, desabilitado
- **Acessível:** Navegação por teclado, ARIA, foco visível, label e mensagem de erro
- **Customizável:** Tamanhos, preenchimento, integração com formulários
- **Feedback visual:** Hover, foco, erro
