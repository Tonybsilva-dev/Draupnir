# Link Component
The `Link` component is used to represent a user, and can display images, icons or initials. It is commonly used in user interfaces to signal the presence of a person or entity.

## Features
- Enables URL navigation through anchor tags.
- Customizable style with additional CSS classes.
- Option to disable the link, preventing navigation.
- Includes accessibility attributes like aria-disabled.
- Supports various content types like text, icons, or images.Fallback support for initials when image is not available
