# Divider

The Divider component is used to separate content and create visual divisions in the interface.

## Description
The Draupnir DS Divider facilitates visual organization, improving readability and information hierarchy.

## Features
- **Customizable:** Thickness, color, and width
- **Horizontal or vertical:** Adapts to context
- **Semantic:** Can contain text or icons, read by screen readers as divider
- **Responsive:** Maintains proportion on different screens
