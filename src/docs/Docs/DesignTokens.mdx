## Overview

Draupnir uses a token architecture based on **CSS Custom Properties**, where all values are defined once in `globals.css` and referenced by TypeScript files. This ensures:

- ✅ **Consistency** between CSS and TypeScript
- ✅ **Dynamic theming** with dark mode support
- ✅ **Optimized performance**
- ✅ **Centralized maintainability**

### Usage in Code

```tsx
import { typography } from '@/tokens';

// Text sizes
typography.text.xs      // var(--text-xs) - 12px
typography.text.sm      // var(--text-sm) - 14px
typography.text.md      // var(--text-md) - 16px
typography.text.lg      // var(--text-lg) - 18px
typography.text.xl      // var(--text-xl) - 20px
typography.text['2xl']  // var(--text-2xl) - 24px
typography.text['4xl']  // var(--text-4xl) - 28px

// Title sizes
typography.title.md     // var(--title-md) - 40px
typography.title.lg     // var(--title-lg) - 48px
typography.title.xl     // var(--title-xl) - 56px

// Font weights
typography.fontWeight.normal    // 400
typography.fontWeight.medium    // 500
typography.fontWeight.semibold  // 600
typography.fontWeight.bold      // 700
```
