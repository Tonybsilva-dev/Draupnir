# Subtitle Page

The Subtitle Page component displays a subtitle, description, and actions at the top of interface sections.

## Description
The Draupnir DS Subtitle Page organizes secondary information and action buttons, improving visual hierarchy.

## Features
- **Accessible:** Semantic typography, visible focus, clear structure
- **Responsive:** Adaptable layout for different screen sizes
- **Customizable:** Allows adding buttons, descriptions, and other elements

## Usage

```tsx
import { SubtitlePage } from '@/components/organisms/SubtitlePage';

<SubtitlePage subtitle="Personal Information" description="Update your data here.">
  <Button variant="outline">Cancel</Button>
  <Button variant="primary">Save</Button>
</SubtitlePage>
```
