## Color Scale

Each color has a scale from 50 to 900, allowing subtle variations for different contexts.

### Primary Scale (Token)

- **50**: rgba(var(--primary), 0.05) - Very light background
- **100**: rgba(var(--primary), 0.1) - Light background
- **200**: rgba(var(--primary), 0.2) - Medium-light background
- **300**: rgba(var(--primary), 0.3) - Medium background
- **400**: rgba(var(--primary), 0.4) - Medium-dark background
- **500**: rgb(var(--primary)) - Main color (Solar Yellow)
- **600**: rgb(var(--hover)) - Hover state
- **700**: rgb(var(--click)) - Click state
- **800**: rgba(var(--primary), 0.8) - Dark color
- **900**: rgba(var(--primary), 0.9) - Very dark color

### Secondary Scale (Token)

- **50**: rgba(76, 175, 80, 0.05) - Very light background
- **100**: rgba(76, 175, 80, 0.1) - Light background
- **200**: rgba(76, 175, 80, 0.2) - Medium-light background
- **300**: rgba(76, 175, 80, 0.3) - Medium background
- **400**: rgba(76, 175, 80, 0.4) - Medium-dark background
- **500**: rgb(var(--secondary)) - Main color (Bluish Gray)
- **600**: rgba(60, 140, 64, 1) - Dark secondary
- **700**: rgba(45, 105, 48, 1) - Darker secondary
- **800**: rgba(30, 70, 32, 1) - Dark color
- **900**: rgba(15, 35, 16, 1) - Very dark color

### Gray Scale

- **50**: #f8fafc - Very light gray
- **100**: #f1f5f9 - Light gray
- **200**: #e2e8f0 - Medium-light gray
- **300**: #cbd5e1 - Medium gray
- **400**: #94a3b8 - Medium-dark gray
- **500**: #64748b - Medium gray
- **600**: #475569 - Dark gray
- **700**: #334155 - Darker gray
- **800**: #1e293b - Very dark gray
- **900**: #0f172a - Darkest gray

## Usage in Code

### Import

```tsx
import { colors } from '@/tokens';
```

### Usage Examples

```tsx
// Cores primárias
const primaryColor = colors.primary[500];  // rgb(var(--primary))
const primaryLight = colors.primary[100];  // rgba(var(--primary), 0.1)
const primaryDark = colors.primary[700];   // rgb(var(--click))

// Cores secundárias
const secondaryColor = colors.secondary[500];  // rgb(var(--secondary))
const secondaryLight = colors.secondary[100];  // rgba(76, 175, 80, 0.1)
const secondaryDark = colors.secondary[700];   // rgba(45, 105, 48, 1)

// Semantic colors
const successColor = colors.success[500];  // #3ac922
const errorColor = colors.error[500];      // #ff3030
const infoColor = colors.info[500];        // #3b82f6
const warningColor = colors.warning[500];  // #f59e0b

// Neutral colors
const grayLight = colors.gray[100];        // #f1f5f9
const grayMedium = colors.gray[500];       // #64748b
const grayDark = colors.gray[800];         // #1e293b
```

### In Components

```tsx
const MyComponent = () => {
  return (
    <div style={{

      color: 'white',

};
```

## Usage Guidelines

### Primary Colors (Solar Yellow)

- **500**: Use for main elements (buttons, links)
- **600**: Use for hover states
- **700**: Use for active/click states
- **100-200**: Use for subtle backgrounds
- **800-900**: Use for text on light backgrounds

### Secondary Colors (Bluish Gray)

- **500**: Use for secondary elements (headers, navigation)
- **600-700**: Use for hover and active states
- **100-200**: Use for subtle backgrounds
- **800-900**: Use for text on light backgrounds

### Semantic Colors

- **Success**: Successful operations, confirmations
- **Error**: Errors, failed validations, critical alerts
- **Warning**: Warnings, actions that require attention
- **Info**: Information, tips, neutral states

### Neutral Colors

- **50-100**: Very light backgrounds
- **200-300**: Borders and separators
