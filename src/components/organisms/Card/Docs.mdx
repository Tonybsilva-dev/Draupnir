# Card

O componente Card agrupa informações de forma visual e minimalista.

## Features

- **Composto:** Header, Content, Image, Action
- **Variantes:** default, elevated, outlined, flat
- **Tamanhos:** sm, md, lg
- **Acessível:** ARIA, navegação por teclado
- **Responsivo:** Adapta-se a diferentes tamanhos de tela

## Uso

```tsx
import { Card, CardHeader, CardContent, CardImage, CardAction } from "@/components/organisms/Card";

<Card>
  <Card.Image src="https://images.unsplash.com/photo-1506744038136-46273834b3fb?w=600&q=80" alt="Produto Premium" aspectRatio="square" />
  <Card.Header>
    <Typography size="xl" className="font-semibold">Produto Premium</Typography>
  </Card.Header>
  <Typography size="sm" className="text-gray-500 mb-2">Categoria: Eletrônicos</Typography>
  <Card.Content>
    <p className="text-gray-700">Descrição detalhada do produto e seus benefícios.</p>
  </Card.Content>
  <Card.Action size="full">Comprar Agora</Card.Action>
</Card>
```
