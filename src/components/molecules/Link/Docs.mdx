# Link

The Link component is used for navigation between pages and redirect actions.

## Description
The Draupnir DS Link ensures accessibility, consistent styles, and router integration.

## Features
- **Accessible:** ARIA support, keyboard navigation, visible focus
- **Customizable:** Color variations, state, and disabled
- **Integration:** Compatible with routers (Next.js, React Router)
- **Visual feedback:** Hover, focus, and active states
