# Pagination

The Pagination component enables navigation between pages of content, supporting accessibility and custom controls.

## Description
The Draupnir DS Pagination provides accessible navigation for multi-page content, with support for page numbers, first/last controls, and keyboard navigation.

## Features
- **Page navigation:** Next, previous, first, last controls
- **Page numbers:** Show/hide page numbers
- **Customizable:** Total pages, current page, showFirstLast, showNumbers
- **Accessibility:** ARIA labels, keyboard navigation
- **Controlled:** Fully controlled current page state
