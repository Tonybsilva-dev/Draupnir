## Color Scale

Each color has a scale from 50 to 900, allowing subtle variations for different contexts.

### Primary Scale (Solar Yellow)

- **50**: rgba(255, 214, 0, 0.05) - Very light background
- **100**: rgba(255, 214, 0, 0.1) - Light background
- **200**: rgba(255, 214, 0, 0.2) - Medium-light background
- **300**: rgba(255, 214, 0, 0.3) - Medium background
- **400**: rgba(255, 214, 0, 0.4) - Medium-dark background
- **500**: #FFD600 - Main color (Solar Yellow)
- **600**: #FFEB3B - Hover state
- **700**: #FFC228 - Click state
- **800**: rgba(255, 214, 0, 0.8) - Dark color
- **900**: rgba(255, 214, 0, 0.9) - Very dark color

### Secondary Scale (Bluish Gray)

- **50**: rgba(58, 79, 99, 0.05) - Very light background
- **100**: rgba(58, 79, 99, 0.1) - Light background
- **200**: rgba(58, 79, 99, 0.2) - Medium-light background
- **300**: rgba(58, 79, 99, 0.3) - Medium background
- **400**: rgba(58, 79, 99, 0.4) - Medium-dark background
- **500**: #3A4F63 - Main color (Bluish Gray)
- **600**: #2d3f4f - Dark secondary
- **700**: #1f2b36 - Darker secondary
- **800**: rgba(58, 79, 99, 0.8) - Dark color
- **900**: rgba(58, 79, 99, 0.9) - Very dark color

### Gray Scale

- **50**: #f8fafc - Very light gray
- **100**: #f1f5f9 - Light gray
- **200**: #e2e8f0 - Medium-light gray
- **300**: #cbd5e1 - Medium gray
- **400**: #94a3b8 - Medium-dark gray
- **500**: #64748b - Medium gray
- **600**: #475569 - Dark gray
- **700**: #334155 - Darker gray
- **800**: #1e293b - Very dark gray
- **900**: #0f172a - Darkest gray

## Usage in Code

### Import

```tsx
import { colors } from '@/tokens';
```

### Usage Examples

```tsx
// Primary colors (Solar Yellow)
const primaryColor = colors.primary[500];  // #FFD600
const primaryLight = colors.primary[100];  // rgba(255, 214, 0, 0.1)
const primaryDark = colors.primary[700];   // #FFC228

// Secondary colors (Bluish Gray)
const secondaryColor = colors.secondary[500];  // #3A4F63
const secondaryLight = colors.secondary[100];  // rgba(58, 79, 99, 0.1)
const secondaryDark = colors.secondary[700];   // #1f2b36

// Semantic colors
const successColor = colors.success[500];  // #3ac922
const errorColor = colors.error[500];      // #ff3030
const infoColor = colors.info[500];        // #3b82f6
const warningColor = colors.warning[500];  // #f59e0b

// Neutral colors
const grayLight = colors.gray[100];        // #f1f5f9
const grayMedium = colors.gray[500];       // #64748b
const grayDark = colors.gray[800];         // #1e293b
```

### In Components

```tsx
const MyComponent = () => {
```

## Usage Guidelines

### Primary Colors (Solar Yellow)

- **500**: Use for main elements (buttons, links)
- **600**: Use for hover states
- **700**: Use for active/click states
- **100-200**: Use for subtle backgrounds
- **800-900**: Use for text on light backgrounds

### Secondary Colors (Bluish Gray)

- **500**: Use for secondary elements (headers, navigation)
- **600-700**: Use for hover and active states
- **100-200**: Use for subtle backgrounds
- **800-900**: Use for text on light backgrounds

### Semantic Colors

- **Success**: Successful operations, confirmations
- **Error**: Errors, failed validations, critical alerts
- **Warning**: Warnings, actions that require attention
- **Info**: Information, tips, neutral states

### Neutral Colors

- **50-100**: Very light backgrounds
- **200-300**: Borders and separators
