# Title Page

O componente Title Page exibe o título principal e uma descrição no topo da página.

## Descrição
O Title Page do Draupnir DS destaca o contexto da página, melhorando a hierarquia visual e a acessibilidade.

## Recursos
- **Acessível:** Usa tags semânticas, foco visível, tipografia clara
- **Customizável:** Permite título e descrição
- **Responsivo:** Adapta-se a diferentes tamanhos de tela
