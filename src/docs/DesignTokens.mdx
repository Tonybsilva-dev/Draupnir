# Design Tokens

Os design tokens são os elementos fundamentais do **Draupnir Design System**. Eles definem as propriedades visuais que garantem consistência, acessibilidade e escalabilidade em toda a aplicação.

## Visão Geral

O Draupnir utiliza uma arquitetura de tokens baseada em **CSS Custom Properties**, onde todos os valores são definidos uma única vez no `globals.css` e referenciados pelos arquivos TypeScript. Isso garante:

- ✅ **Consistência** entre CSS e TypeScript
- ✅ **Tema dinâmico** com suporte a modo escuro
- ✅ **Performance** otimizada
- ✅ **Manutenibilidade** centralizada

### Escala de Cores

Cada cor possui uma escala de 50 a 900, permitindo variações sutis para diferentes contextos:

```tsx
import { colors } from '@/tokens';

// Cores primárias
colors.primary[50]   // rgba(var(--primary), 0.05) - Background muito claro
colors.primary[100]  // rgba(var(--primary), 0.1)  - Background claro
colors.primary[200]  // rgba(var(--primary), 0.2)  - Background médio-claro
colors.primary[300]  // rgba(var(--primary), 0.3)  - Background médio
colors.primary[400]  // rgba(var(--primary), 0.4)  - Background médio-escuro
colors.primary[500]  // rgb(var(--primary))        - Cor principal
colors.primary[600]  // rgb(var(--hover))          - Cor hover
colors.primary[700]  // rgb(var(--click))          - Cor click
colors.primary[800]  // rgba(var(--primary), 0.8)  - Cor escura
colors.primary[900]  // rgba(var(--primary), 0.9)  - Cor muito escura
```

### Uso em Código

```tsx
import { typography } from '@/tokens';

// Tamanhos de texto
typography.text.xs      // var(--text-xs) - 12px
typography.text.sm      // var(--text-sm) - 14px
typography.text.md      // var(--text-md) - 16px
typography.text.lg      // var(--text-lg) - 18px
typography.text.xl      // var(--text-xl) - 20px
typography.text['2xl']  // var(--text-2xl) - 24px
typography.text['4xl']  // var(--text-4xl) - 28px

// Tamanhos de título
typography.title.md     // var(--title-md) - 40px
typography.title.lg     // var(--title-lg) - 48px
typography.title.xl     // var(--title-xl) - 56px

// Pesos de fonte
typography.fontWeight.normal    // 400
typography.fontWeight.medium    // 500
typography.fontWeight.semibold  // 600
typography.fontWeight.bold      // 700
```

import { colors, spacing, typography, shadows } from '@/tokens';

const Card = ({ children, variant = 'default' }) => {

import { getColor, getSpacing, getTextSize, getShadow } from '@/tokens';
