# Text Block

The Text Block component is used to display text blocks with title and description, facilitating information organization.

## Description
The Draupnir DS Text Block highlights important texts, section titles, and descriptions in a clear and accessible manner.

## Features
- **Accessible:** Semantic typography, visible focus, clear structure
- **Customizable:** Allows title, description, and children
- **Responsive:** Adapts to different screen sizes

## Usage

```tsx
import { TextBlock } from '@/components/atoms/Text';

<TextBlock title="Title" type="primary">
  Description or content of the text block.
</TextBlock>
```
