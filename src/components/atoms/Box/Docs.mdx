# Box

O componente Box é um container versátil para agrupar e estruturar conteúdos na interface.

## Descrição
O Box do Draupnir DS facilita a organização visual, aplicação de espaçamentos, fundos e bordas.

## Recursos
- **Versátil:** Pode ser usado para cards, seções, agrupamentos
- **Customizável:** Suporte a bordas, arredondamento e preenchimento
- **Temático:** Variações de cor e fundo
- **Responsivo:** Adapta-se a diferentes tamanhos de tela
