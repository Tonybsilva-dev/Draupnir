# Notice Component
The `Notice` component is used to represent a user, and can display images, icons or initials. It is commonly used in user interfaces to signal the presence of a person or entity.

## Features
- Variety of Types: Includes alert, success, error, info, and more to suit different contexts.
- Icon Display: Automatically shows relevant icons for clear communication.
- Custom Content: Supports adding custom elements or messages.
- Dismissible: Features an optional close button for user interaction.
- Keyboard Friendly: Can be closed with the Escape key for accessibility.
- Tailwind Styled: Offers easy customization with Tailwind CSS.
