# Divider

O componente Divider é utilizado para separar conteúdos e criar divisões visuais na interface.

## Descrição
O Divider do Draupnir DS facilita a organização visual, melhorando a leitura e a hierarquia das informações.

## Recursos
- **Customizável:** Espessura, cor e largura
- **Horizontal ou vertical:** Adapta-se ao contexto
- **Semântico:** Pode conter texto ou ícones
- **Responsivo:** Mantém proporção em diferentes telas
