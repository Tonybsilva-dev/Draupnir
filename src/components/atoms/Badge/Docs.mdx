# Badge

O componente Badge é utilizado para exibir informações de status, contadores ou rótulos de forma compacta e visualmente destacada.

## Descrição
O Badge do Draupnir DS permite sinalizar estados, categorias ou quantidades de maneira clara e acessível.

## Recursos
- **Variações visuais:** Primary, Success, Warning e Danger
- **Compacto:** Design minimalista que não interfere no layout
- **Semântico:** Cores que comunicam significado (verde=sucesso, vermelho=erro)
- **Flexível:** Aceita qualquer conteúdo como children
- **Acessibilidade:** Alto contraste para boa legibilidade
