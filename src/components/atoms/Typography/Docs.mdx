# Typography

The Typography component is used to style texts, titles, and paragraphs consistently.

## Description
The Draupnir DS Typography ensures visual standardization, hierarchy, and accessibility in all interface texts.

## Features
- **Accessible:** Semantic typography, visible focus, clear sizes and weights
- **Customizable:** Size, weight, and color variations
- **Consistent:** Maintains visual pattern throughout the application

## Usage

```tsx
import { Typography } from '@/components/atoms/Typography';

<Typography size="lg" variant="primary">
  Large and highlighted text
</Typography>
```
