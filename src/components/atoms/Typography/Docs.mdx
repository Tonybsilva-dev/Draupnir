# Typography

O componente Typography é utilizado para estilizar textos, títulos e parágrafos de forma consistente.

## Descrição
O Typography do Draupnir DS garante padronização visual, hierarquia e acessibilidade em todos os textos da interface.

## Recursos
- **Acessível:** Tipografia semântica, foco visível, tamanhos e pesos claros
- **Customizável:** Variações de tamanho, peso e cor
- **Consistente:** Mantém padrão visual em toda a aplicação
