# Sidebar

The Sidebar component provides a vertical navigation menu with logo, navigation links, notification badge, and logout action. It is fully styled with Draupnir tokens and ready for use in dashboards and applications.

## Description
The Draupnir DS Sidebar is a vertical navigation component for layouts, featuring logo/avatar, navigation links, notification badges, and quick actions.

## Features
- **Vertical navigation:** For dashboards and apps
- **Logo/avatar:** Top branding
- **Navigation links:** With icons and badges
- **Logout action:** Quick access at the bottom
- **Customizable:** Uses Draupnir tokens for full theming

## Accessibility
- Uses semantic HTML for navigation and actions
- All links and buttons are keyboard accessible
- Uses proper color contrast and focus states

## Styling & Animation
- Uses Draupnir tokens for color, spacing, border, and typography
- Add this to your globals.css for hover effect:

```css
.sidebar-link:hover,
.sidebar-link:focus {
  background: rgb(var(--divider-light));
  color: rgb(var(--primary));
  text-decoration: none;
}
```
