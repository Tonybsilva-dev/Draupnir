# Button

O componente Button é utilizado para acionar ações e interações em interfaces.

## Descrição
O Button do Draupnir DS oferece diferentes estilos visuais, estados e acessibilidade, sendo essencial para qualquer aplicação web.

## Recursos
- **Variações visuais:** Primary, Outline, Ghost e Danger
- **Tamanhos:** Small, Default, Large e Full
- **Feedback visual:** Hover, foco e clique
- **Estado de carregamento:** Spinner integrado
- **Customização:** Tamanhos, preenchimento e variantes
- **Acessibilidade:** HTML semântico e suporte a ARIA
