# Typography

The Typography component is used to style texts, titles, and paragraphs consistently using design system tokens.

## Description
The Draupnir DS Typography ensures visual standardization, hierarchy, and accessibility in all interface texts, using only design system tokens.

## Features
- **Accessible:** Semantic typography, visible focus, clear sizes and weights
- **Customizable:** Size, weight, and color variations through tokens
- **Consistent:** Maintains visual pattern throughout the application
- **Tokenized:** Uses only design system tokens, no hardcoded values

## Usage

```tsx
import { Typography } from '@/components/atoms/Typography';

// Basic text
<Typography size="lg" variant="primary">
  Large and highlighted text
</Typography>

// Semantic heading
<Typography semantic="heading" level={1} size="title1">
  Main Title
</Typography>

// Text with custom styles
<Typography 
  size="md" 
  variant="secondary"
  style={{ backgroundColor: '#f0f0f0', padding: '16px' }}
>
  Text with custom styles
</Typography>
```

## Design System Tokens

The Typography component uses the following design system tokens:

### Colors
- `colors.text.primary` - Primary text color
- `colors.text.secondary` - Secondary text color  
- `colors.text.tertiary` - Tertiary text color

### Typography
- `typography.fontWeight.normal` - Font weight
- `typography.lineHeight.normal` - Line height

### Font Sizes
- `xs`: 12px
- `sm`: 14px
- `md`: 16px
- `lg`: 18px
- `xl`: 20px
- `2xl`: 24px
- `4xl`: 32px
- `title1`: 28px
- `title2`: 24px
- `title3`: 20px
