# Button

The Button component is used to trigger actions and interactions in interfaces.

## Description
The Draupnir DS Button offers different visual styles, states, and accessibility, being essential for any web application.

## Features
- **Visual variations:** Primary, Secondary, Outline, Ghost, and Danger
- **Sizes:** Small, Default, Large, and Full
- **Visual feedback:** Hover, focus, and click states
- **Loading state:** Integrated spinner
- **Customization:** Sizes, padding, and variants
- **Accessibility:** Semantic HTML and ARIA support
