# Link

O componente Link é utilizado para navegação entre páginas e ações de redirecionamento.

## Descrição
O Link do Draupnir DS garante acessibilidade, estilos consistentes e integração com roteadores.

## Recursos
- **Acessível:** Suporte a ARIA e navegação por teclado
- **Customizável:** Variações de cor, estado e desabilitado
- **Integração:** Compatível com roteadores (Next.js, React Router)
- **Feedback visual:** Hover, foco e estados ativos
