# ButtonIcon

## Description
ButtonIcon is a minimalist button that displays only an icon, ideal for quick actions, toolbars, and cards. It offers full accessibility via `aria-label` and supports multiple variants and sizes.

## Features
- Icon only, no text
- Accessibility support via `aria-label`
- Variants: ghost, outline, primary, danger
- Sizes: sm, md, lg

## Usage
```tsx

import { Plus } from 'lucide-react';

<ButtonIcon icon={<Plus />} aria-label="Add" />
```
