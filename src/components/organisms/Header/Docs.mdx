# Header

The Draupnir DS Header organizes navigation, logo, and user area in a responsive and accessible way.

## Overview

The Draupnir DS Header allows you to create consistent navigation headers with complete accessibility and theme support.

## Features

- **Accessible:** Keyboard navigation, ARIA, visible focus
- **Responsive:** Adapts to different screen sizes
- **Composable:** Logo, center navigation, and user area on the right
- **Customizable:** Allows adding menus, dropdowns, and avatar
