# Dropdown

The Dropdown component provides an interactive and accessible dropdown menu.

## Overview

The Draupnir DS Dropdown allows you to create contextual menus with complete accessibility and theme support.

## Features

- **Accessible:** Complete ARIA support, keyboard navigation, visible focus
- **Themed:** Light and dark theme support
- **Animated:** Smooth enter and exit transitions
- **Flexible:** Component-based composition (Trigger, Content, Item)
- **Responsive:** Adapts to different screen sizes

## Usage

```tsx
import { Dropdown } from '@/components/organisms/Dropdown';
import { Button } from '@/components/atoms/Button';

<Dropdown>
  <Dropdown.Trigger asChild>
    <Button variant="outline">Open Menu</Button>
  </Dropdown.Trigger>
  <Dropdown.Content>
    <Dropdown.Item>Option 1</Dropdown.Item>
    <Dropdown.Item>Option 2</Dropdown.Item>
  </Dropdown.Content>
</Dropdown>
```
