# Title Page

The Title Page component displays the main title and description at the top of the page.

## Description
The Draupnir DS Title Page highlights the page context, improving visual hierarchy and accessibility.

## Features
- **Accessible:** Uses semantic tags, visible focus, clear typography
- **Customizable:** Allows title and description
- **Responsive:** Adapts to different screen sizes
