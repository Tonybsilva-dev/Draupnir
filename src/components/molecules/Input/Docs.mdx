# Input

The Input component is used to capture and submit user data.

## Description
The Draupnir DS Input offers text fields with prefix, error states, disabled state, and form integration.

## Features
- **Prefix:** Support for icons or text before the field
- **States:** Normal, error, disabled
- **Accessible:** Keyboard navigation, ARIA, visible focus, label and error message
- **Customizable:** Sizes, padding, form integration
- **Visual feedback:** Hover, focus, error

## Usage

```tsx
import { Input } from '@/components/molecules/Input';

<Input.Root label="Name" errorMessage="Name is required">
  <Input.Prefix>
    <Search className="w-5 h-5" />
  </Input.Prefix>
  <Input.Control placeholder="Enter your name" />
</Input.Root>
```
