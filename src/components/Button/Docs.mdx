# Button
This is a button molecule.

## Features
- Offers multiple visual styles: Primary, Outline, and Ghost.
- Provides visual feedback with hover, focus, and active states.
- Includes a loading state with an integrated spinner.
- Customizable in terms of size and padding.
- Accessible with semantic HTML and ARIA attributes.
