# EmptyState

## Description
EmptyState is a component to display empty states in lists, tables, or pages. It composes icon, title, description, and action in an accessible and customizable visual block.

## Features
- Customizable icon, title, description, and action
- Uses lucide-react icons
- Fully accessible

## Usage
```tsx

<EmptyState />
<EmptyState title="No results" description="Please try again later." />
```
