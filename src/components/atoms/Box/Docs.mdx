# Box

The Box component is a versatile container for grouping and structuring content in the interface.

## Description
The Draupnir DS Box facilitates visual organization, application of spacing, backgrounds, and borders.

## Features
- **Versatile:** Can be used for cards, sections, groupings
- **Customizable:** Support for borders, rounding, and padding
- **Thematic:** Color and background variations
- **Responsive:** Adapts to different screen sizes
- **Accessible:** Can be used as semantic container, doesn't interfere with keyboard navigation
