# Avatar

The Avatar component is used to display user images, icons, or initials in a square and responsive format.

## Description
The Draupnir DS Avatar is ideal for representing profiles, user lists, and authentication areas.

## Features
- **Sizes:** xs, sm, md, lg
- **Fallback:** Default icon when no image is available
- **Accessible:** Alt description support for screen readers, visible focus when navigating with keyboard, fallback icon
- **Interactive:** Hover and focus states
- **Flexible:** Accepts images or default icon
