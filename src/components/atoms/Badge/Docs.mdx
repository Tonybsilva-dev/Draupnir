# Badge

The Badge component is used to display status information, counters, or labels in a compact and visually highlighted format.

## Description
The Draupnir DS Badge allows signaling states, categories, or quantities in a clear and accessible manner.

## Features
- **Visual variations:** Primary, Success, Warning, and Danger
- **Compact:** Minimalist design that doesn't interfere with layout
- **Semantic:** Colors that communicate meaning (green=success, red=error)
- **Flexible:** Accepts any content as children
- **Accessible:** High contrast, readable by screen readers, semantic colors
