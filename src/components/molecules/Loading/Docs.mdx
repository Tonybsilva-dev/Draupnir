# Loading

O componente Loading exibe um indicador animado de carregamento.

## Descrição
O Loading do Draupnir DS utiliza animações suaves para indicar processos em andamento, centralizando o feedback visual.

## Recursos
- **Acessível:** Pode ser lido por leitores de tela (aria-label), foco visível
- **Animado:** Efeito ping e pulse para melhor percepção
- **Centralizado:** Sempre centraliza o indicador no container
- **Customizável:** Pode ser usado em diferentes contextos
