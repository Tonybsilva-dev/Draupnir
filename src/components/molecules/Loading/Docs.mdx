# Loading

The Loading component displays an animated loading indicator.

## Description
The Draupnir DS Loading uses smooth animations to indicate ongoing processes, centralizing visual feedback.

## Features
- **Accessible:** Can be read by screen readers (aria-label), visible focus
- **Animated:** Ping and pulse effect for better perception
- **Centered:** Always centers the indicator in the container
- **Customizable:** Can be used in different contexts
