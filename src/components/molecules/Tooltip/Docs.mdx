# Tooltip

Accessible tooltip component for hints, descriptions, and extra information on hover or focus.

## Example
<Canvas withToolbar>
  <Story of={TooltipStories.Default} />
</Canvas>

## Usage

```tsx
import { Tooltip } from '@/components/molecules/Tooltip';

<Tooltip content="Tooltip text">
  <button>Hover me</button>
</Tooltip>
```
