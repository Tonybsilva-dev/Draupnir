# DatePicker

The DatePicker component allows users to select a date from a calendar or input field, supporting accessibility and multiple formats.

## Description
The Draupnir DS DatePicker provides an accessible, interactive calendar, supports date ranges, and allows for custom formats, making it ideal for forms and scheduling interfaces.

## Features
- **Calendar selection:** Interactive calendar popup
- **Manual input:** Accepts direct date typing
- **Date range:** minDate and maxDate support
- **Formats:** "dd/MM/yyyy" and "yyyy-MM-dd"
- **Accessibility:** ARIA labels, keyboard navigation
- **Customizable:** Placeholder, disabled, required
